# FalconMindSDK 示例05：Flow执行器

## 概述

本示例演示如何使用FalconMindSDK的FlowExecutor进行流程调度执行。FlowExecutor是SDK中的流程调度器，负责管理Pipeline的执行和状态转换。

## 测试的SDK API

本示例测试了以下核心SDK API：

### FlowExecutor核心API
| API | 功能 | 验证 |
|-----|------|------|
| `FlowExecutor::addFlow()` | 添加流程 | ✅ |
| `FlowExecutor::execute()` | 执行流程 | ✅ |
| `FlowExecutor::stop()` | 停止执行 | ✅ |
